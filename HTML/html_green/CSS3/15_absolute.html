Position relative

        static : 기본값
        relative : 요소의 원래 위치를 기준으로 배치
        absolute : 가까운 부모 중 position 속성의 값이 static이 아닌 요소를 기준으로 배치
        fixed : 브라우저 윈도우를 기준으로 배치
        sticky : 스크롤이 생기면 지정한 위치에서 고정됨
